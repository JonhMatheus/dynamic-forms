# Dynamic Forms

Dynamic Forms é um aplicativo que permite a criação e gerenciamento de formulários dinâmicos. Os usuários podem criar, preencher e enviar respostas para formulários personalizados.

## Requisitos

### Android
- SDK 21 ou superior

### iOS
- Versão 17 ou superior
